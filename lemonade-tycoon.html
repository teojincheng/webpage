Lemonade Tycoon Clone

The web app is located at: https://vigorous-hugle-ee108f.netlify.com/

Lemonade Tycoon is a business simulation game originally released in year 2002.
The video above shows the gameplay of the imitation that I have developed in year 2020.

Below is a description of the process that takes place in the clone version:

  • The game has two game days
  • At the start of each day: the player determines how much of the raw ingredients he/she wants to buy.
  • The player then determines how much of each ingredient he/she want to use to make 1 cup of lemonade
  • Player determines the selling price for each cup of lemonade

While doing this project, I make use of Cypress to do some form of end to end testing

The front end of the app is created using React.js.The backend of the app uses Express.js with MongoDB.The react app hosted on Netlify makes API calls to the Express server and MongoDB hosted on Heroku.
